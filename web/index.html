Research Track

Software Development

Software development is the series of activities that builds up software product. Generally speaking, there are several phases in software development, such as requirement, design, implementation and maintenance…etc. We follow the principles of software development to deal with DBSE projects. Sometimes problems have occurred during project development, they probably would be good research motivations to be accomplished. Use Case and Refactoring are both the frequent activities that we often decide to carry out.

Programming Education

We are experienced lab. for computer science education. Computer Science Course students are potential software engineers in the future. On the course, students realize software development by programming homework practicing. Teaching Assist(TA) designs homework assignment and evaluate the programming homework. Course students writing programs to match assignment, upload program and document, and waiting for evaluation result. In this process, we are interested in program learning improvement, rapid program evaluation response, and automation of TA management process.

Information Processing

Information processing concerns with gathering, manipulating, storing, retrieving, and classifying recorded information. Currently, the problems we have concerned are about information visualization, information retrieval and human computer interaction improvement.

Open Source

We investigate open source community activities and collect project activities information from projects open source foundry. We are interested in factors of success open source project and process of open source software development. With these issues, we are devoted to model source development process and assist open source development process.
